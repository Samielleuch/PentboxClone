# Installation

## Development
Clone this repo
`git clone git@github.com:Samielleuch/PentboxClone.git`

Create a python3 virtual environement
`python3 -m venv venv`

Install project dependencies 
`pip install -r requirements.txt`

## How to use 
`git clone git@github.com:Samielleuch/PentboxClone.git`

Create a python3 virtual environement
`cd PentboxClone`
<br>
`python3 CLIEngine.py`
<br>
or
<br>
 
`chmod +x CLIEngine.py`
<br/>
`./CLIEngine.py`
